HSLIDER

<< Horizontal slider >>

Using jQuery 3.5.1 (needed).


See a demonstration

- Slide any element left/right/right-left;
- Slide on click, grab or swipe on mobile(left/right);
- Responsive: adjustable to any screen width;
- Change style on .css file;
